# Heart Disease Analysis Using Tableau

## Project Overview
This project analyzes heart disease data using Tableau. Interactive dashboards and stories were created to identify trends and insights related to heart disease.

## Tools Used
- Tableau Desktop
- Tableau Public

## Dataset
- heart_2020_cleaned.csv

## Features
- Interactive Dashboards
- Filters
- Story
- Data Visualization
